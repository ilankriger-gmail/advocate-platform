# Documentar componentes UI com exemplos de props

## Overview

Os componentes UI em src/components/ui/ (Button, Input, Card, Modal, etc.) possuem interfaces TypeScript mas nao tem documentacao JSDoc explicando o proposito de cada prop, valores possiveis, ou exemplos de uso.

## Rationale

A biblioteca de componentes e usada em toda a aplicacao. Documentacao clara de props acelera o desenvolvimento e garante uso consistente. Componentes como Button tem 6+ props (variant, size, isLoading, icons) que precisam ser explicadas.
