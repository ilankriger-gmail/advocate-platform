# Criar guia de troubleshooting para erros comuns

## Overview

O projeto depende de multiplas configuracoes externas (Supabase, variaveis de ambiente, APIs) mas nao existe documentacao de resolucao de problemas. Erros como 'API Gemini nao configurada' aparecem no codigo mas sem guia de como resolver.

## Rationale

Desenvolvedores frequentemente enfrentam erros de configuracao que consomem horas para resolver. Um guia de troubleshooting com erros comuns e solucoes economiza tempo significativo.
